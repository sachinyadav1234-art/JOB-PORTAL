# JOB-PORTAL

An advanced tech recruitment hub and preparation platform tailored specifically for BTech IT and Computer Science freshers/graduates in India.

## Features
- **Official Verified Jobs**: Live web crawler pulling direct application listings from official corporate boards (Lever, Greenhouse, Workday, Ashby) without middleman spam.
- **Mass Drives & Job Fairs Tracker**: Track ongoing and upcoming mass hiring drives (TCS NQT, Infosys, Wipro Elite, Cognizant, Accenture) and tech job fairs.
- **AI ATS Resume Reviewer**: Scan and optimize resume formatting, highlighting missing core IT curriculum concepts (DBMS, Operating Systems, Computer Networks) and modern development tools.
- **AI Mock Technical Interviews**: Live mock sessions covering DSA, programming syntax, and academic subjects with instant AI grading (scores out of 10) and sample answers.
- **Modern Dashboard & UI**: Stunning futuristic glassmorphism interface styled with Tailwind CSS.

## Tech Stack
- **Frontend**: React, Tailwind CSS, Vite, Axios, React Router, React Hot Toast
- **Backend**: Node.js, Express, MongoDB (Mongoose)
- **AI/APIs**: Groq Cloud SDK (Llama 3.3), Serper Search API
